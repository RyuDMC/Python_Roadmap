This is a basic Expense Tracker CLI.

To execute the CLI run the main.py script and use the right indications after.

#Commands

1/ add -> add an expense

2/ update -> update an expense

3/ delete -> delete an expense

4/ list -> show a list of the expenses

5/ summary -> show a summary of expenses

#Second command

--description -> add the description of added expense

--amount -> add the amount of added expense

--ID -> select the ID of required expense

--mount -> select a mount for the summary

#this are some examples:
1/ add --description lunch --amount 300

2/ delete --ID 3

3/ update --ID 3 --amount 200
